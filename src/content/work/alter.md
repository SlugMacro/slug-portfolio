Platform where users create AI agents that livestream on BNB Chain — each agent issues its own token for community trading. Designed the full mobile experience covering agent creation, live streaming with real-time chat, token buy/sell flows with bonding curve mechanics, agent ranking leaderboard, and user portfolio management.

## Challenge

AI agent platforms existed for chat, but none combined live streaming with token economics — users had no way to discover, watch, and invest in AI agents in real time.

## Approach

Designed the end-to-end agent lifecycle: creation flow, livestream with real-time chat, token trading with bonding curve mechanics and PancakeSwap graduation, ranking leaderboard, and user portfolio management.

## Outcome

Shipped on BNB Chain — mobile-first agent creation and trading with live streaming integration.
